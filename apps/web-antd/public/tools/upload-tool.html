图片标准命名打包工具

把美工交付的未规范命名图片、视频按 A+ 或品牌故事规则重命名，并直接生成对应 zip 下载；需要创建飞书任务时，会把当前生成的 zip 上传到飞书多维表格。

本地重命名 生成 ZIP A+ / 品牌故事

基础设置

SPU 用于生成标准文件名；压缩包名称可单独设置，适配样例里的 `573A+.zip`、`423品牌故事.zip`。
输入店铺和 SPU 后可以查询领星在线 Listing 数据。

A+ 素材

按组件添加文件。完整图和轮播共用 A+ 图片组序号；视频包含视频文件和封面，不占用图片组序号。
拖入 A+ 文件后再分配到下方组件 也可以直接在每个文件位选择图片或视频。

品牌故事素材

固定输出 `品牌故事_手机端` 和 `品牌故事_电脑端`；品牌卡媒体资产默认重命名为 SPU-序号。
